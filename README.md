# Student Portal Project - Deliverable B

## Project Overview
This project is a Java-based Student Portal developed using **Test-Driven Development (TDD)**. It demonstrates the implementation of core features while maintaining a professional Git-based workflow.

## Features Implemented
* **User Login:** Secure validation of student credentials.
* **Password Reset:** Logic to handle password recovery for registered students.
* **Transcript Request:** A service for students to request official academic records.

## Technology Stack
* **Language:** Java
* **Testing Framework:** JUnit 5
* **Project Management:** Jira (SCRUM)
* **Version Control:** Git & GitHub

## TDD Process
1. **Red Phase:** Wrote failing JUnit tests for each requirement.
2. **Green Phase:** Implemented minimal code in `LoginService` to pass the tests.
3. **Refactor Phase:** Optimized code using constants and clean code principles.
